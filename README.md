# 🌦️ Weather App

A simple and responsive Weather Forecast Web Application built using HTML, CSS, and JavaScript.  
It fetches real-time weather data using the OpenWeatherMap API and displays weather information based on user input.

---

## 🚀 Features

- 🌍 Real-time weather data
- 🔎 City search functionality
- 📍 City name autocomplete (Geocoding API)
- 🌡️ Temperature, humidity, and "feels like" data
- 🌤️ Weather condition display with icons
- 🎨 Dynamic background based on weather
- ⌨️ Enter key support for search
- 📱 Responsive UI design

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- REST API  
- OpenWeatherMap API  

---

## 🔑 API Used

This project uses the **OpenWeatherMap API**:

- Current Weather API  
- Geocoding API (for city suggestions)

👉 https://openweathermap.org/api
